The game record format has two types of entries. The first type denotes where a new tile has been placed. Each board position is assigned a letter from A to P. If the letter in the record is lowercase, the new tile is a 2. Otherwise it's a 4. The second type of entry is either "u", "r", "d", or "l", representing the move made by the player.
